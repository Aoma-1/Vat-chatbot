# VAT Assistant Chatbot

A simple HTML/JS chatbot powered by OpenAI API that helps answer VAT-related questions in Kenya.

## Features
- Uses GPT-3.5 Turbo
- Calculates VAT percentages
- Gives tax advice and reminders
